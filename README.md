# Capability Evidence Model (CEM)

Capability Evidence Model (CEM) is a research framework for investigating human capability through evidence rather than relying primarily on compressed summaries such as résumés, portfolios, or interviews.

The repository documents the framework's reasoning, architecture, and development as a structured body of work.
